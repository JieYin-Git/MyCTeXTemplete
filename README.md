# MyCTeXTemplete

一个基于ctexart文档类的LaTeX文章模板。

# 文件介绍

- `testctex.tex`模板的主文件
- `bibtest.bib`参考文献库
- `figures`图片文件夹

# 编译方式与编码

需编译5次。顺序为XeLaTeX - makeindex - biber - XeLaTeX - XeLaTeX 。

编码为UTF-8。

建议使用TexLive2022+TexStudio运行。
